lessweb是什么？

lessweb是一个极易上手的python web框架，编写之初受到web.py的启发，其目标是：

  • 支持最新的python版本和最新的python语法
  • 尽量降低学习难度
  • 尽量pythonic

"Django lets you write web apps in Django. Flask lets you write web apps in Flask. Lessweb lets you write web apps in Python."

安装

pip3 install lessweb

lessweb依赖python3.6引入的类型标注语法，所以只支持python3.6.0以上版本。

Hello, world!

from lessweb import Application
def hello():
    return 'Hello, world!'

app = Application()
app.add_get_mapping('/', hello)
app.run()

运行之后，用浏览器访问http://localhost:8080/，就能看到显示"Hello, world!"的页面。

如果想把端口设置为80，可以在app.run()里面设置port=80，也就是：

......
app.add_get_mapping('/', hello)
app.run(port=80)

谁在用lessweb？

  • www.lessweb.cn网站本身就是用lessweb开发的。
  • 秒差距三体平台，是对公众开放的微服务平台，包括：
    • 利用微信开放平台实现的微信常用功能托管系统，可帮助微信开发者提升开发效率。
    • 对阿里云短信进行包装后的短信系统，可提升短信功能开发效率。
  • 第一本编程入门手册，它利用微信公众号回复功能并基于lessweb实现了一个查字典机器人。
  • 曲度美商城，量身定制的枕头。
